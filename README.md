# jsts-rails

[JSTS Topology Suite](https://github.com/bjornharrtell/jsts) packaged for Rails assets pipeline.

## Usage

Add the following to your gemfile:

```ruby
gem "jsts-rails"
```

Add the following directive to your Javascript manifest file (application.js):

```js
//= require jsts
```
